# scaffold-clase17-portfolio-dinamico

## Descripción

Este proyecto es un portafolio dinámico desarrollado utilizando HTML, CSS y JavaScript. El objetivo principal es mostrar una colección de proyectos de manera interactiva y atractiva. El portafolio permite a los usuarios navegar a través de diferentes proyectos, ver detalles específicos de cada uno y acceder a enlaces relacionados.

## Características

- **Diseño Responsivo**: El portafolio se adapta a diferentes tamaños de pantalla, asegurando una experiencia óptima tanto en dispositivos móviles como en computadoras de escritorio.
- **Interactividad**: Los usuarios pueden hacer clic en cada proyecto para ver más detalles, incluyendo una descripción, tecnologías utilizadas y enlaces a repositorios o sitios web relacionados.
- **Animaciones**: Se han implementado animaciones suaves para mejorar la experiencia del usuario al interactuar con los proyectos.
- **Organización**: Los proyectos están organizados de manera clara y accesible, permitiendo a los visitantes encontrar fácilmente la información que buscan.
